# Release Notes v2.6.0

## Italiano
### Panoramica
Tipo di rilascio: minor (v2.6.0). Contiene nuove playground interattive, estese aggiunte di documentazione e immagini, script di supporto per coverage e sincronizzazione immagini, e alcune pulizie di package.json.

### Novita
- Aggiunte playground interattive:
  - Node: selezione/import moduli (NodeModuleSelectorPlayground).
  - Express: ciclo di vita delle sessioni (SessionLifecyclePlayground).
- Nuovi contenuti e file di documentazione per più argomenti (UI Kit, Design System, Algoritmi, API, autenticazione, ecc.).
- Migliorata resa delle immagini nella documentazione e sincronizzazione immagini aggiornata.
- Script per generazione report di coverage aggiunto.

### Correzioni
- Nessuna correzione specifica segnalata nei commit inclusi in questo intervallo.

### Migliorie Tecniche
- Rimosso codice/ script non usato in package.json per una configurazione più pulita.
- Refactor dei playground Chmod e SSH keygen per migliore gestione dello stato e miglioramenti UI.
- Aggiunti/aggiornati script: scripts/docs-sync-images.mjs, scripts/docs-check-nbsp-content.mjs.
- Aggiornamento di package-lock.json correlato alle modifiche di package.json.
- Aggiornate istruzioni personalizzate GPT (file GPT_CUSTOM_DOCS_INSTRUCTIONS.md) — patch.

### Modifiche Incompatibili
- Nessuna incompatibilità esplicita registrata nei commit. Nota: sono stati rimossi script da package.json (vedi Note di Migrazione).

### Note di Migrazione
- Controllare package.json per eventuali script rimossi o modificati che potrebbero influenzare workflow locali o CI.
- Se usate istruzioni GPT personalizzate, verificare GPT_CUSTOM_DOCS_INSTRUCTIONS.md per aggiornamenti.

### Limiti Noti
- Nessun limite specifico documentato nei commit di questo rilascio.

- Interactive playgrounds:
  - Node: module selector (NodeModuleSelectorPlayground).
  - Express: session lifecycle playground (SessionLifecyclePlayground).
- New documentation files across multiple topics (UI Kit, Design System, Algorithms, API, Authentication, etc.).
- Improved image rendering in docs and updated image synchronization scripts.
- Coverage report generation script added.

### Bug Fixes
- No explicit bug fixes recorded in the commits for this range.
