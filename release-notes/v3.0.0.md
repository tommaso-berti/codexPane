# Release Notes v3.0.0

## Italiano

### Panoramica
Versione major (v3.0.0) che introduce miglioramenti all'esperienza utente e alle prestazioni della webapp: gestione temi e preferenze utente, indicizzazione per topic, miglioramenti al caricamento della documentazione e alla ricerca, miglioramenti alla barra laterale/SectMenu e aggiornamenti alla documentazione di deployment. Confronta: https://github.com/tommaso-berti/codexPane/compare/v2.6.0...v3.0.0

### Novita
- Migliorata la gestione del tema e le preferenze utente: palette accent e supporto per "reading time" (commit 48682ca).
- Aggiunto un indice per ogni topic per ottimizzare le prestazioni della webapp (commit 76838fd).
- Migliorato il caricamento della documentazione e la funzionalità di ricerca tramite manifest dei topic e gestione di stato migliorata (commit 5b1063b).
- Estesa la SectMenu con funzionalità di ricerca e persistenza delle opzioni della sidebar (commit ab14cc2).
- Aggiornata la documentazione di deployment e aggiunto uno script unificato per il deploy su VPS (commit 3db5156).

### Correzioni
- Nessuna correzione esplicita segnalata nei commit inclusi in questo intervallo.

### Migliorie Tecniche
- Generazione/aggiornamento di manifest e indici dei contenuti: molti file sotto src/content/docs-topics/*.generated.json e src/content/docs-manifest.generated.json sono stati aggiornati.
- Miglioramenti alla ricerca e agli hook: modifiche in src/features/search/*, src/features/search/useMiniSearchFromDocs.js, src/features/docs/useDocContent.js.
- State management e preferenze utente: aggiornamenti in src/store/uiPrefsSlice.js e src/store/store.js.
- Componenti UI aggiornati: modifiche a SectMenu, Sidebar, SettingsDrawer, DocContent, Header, Layout, App (file sotto src/components/*).
- Script di deployment aggiunto: scripts/deploy-vps-unified.sh; documentazione VPS aggiornata sotto src/content/vps/*.

### Modifiche Incompatibili
- Nei metadati dei commit non sono riportate modifiche incompatibili esplicite. Si raccomanda di verificare i cambiamenti nei file generati sotto src/content/ se vengono consumati da processi esterni.

### Note di Migrazione
- Se la vostra integrazione dipende dai file generati (es. src/content/docs-manifest.generated.json, src/content/docs-index.generated.json o i singoli docs-topic.*.generated.json), aggiornate i processi che li consumano.
- Per il deploy su VPS, è disponibile il nuovo script scripts/deploy-vps-unified.sh e la documentazione aggiornata in src/content/vps/*: verificate e usate il nuovo flusso documentato.
- Se usate preferenze utente salvate (UI prefs), considerate di verificare compatibilità con le modifiche in src/store/uiPrefsSlice.js.

### Limiti Noti
- Nessun limite noto esplicitamente documentato nei commit inclusi in questo rilascio.

## English

### Overview
Major release v3.0.0 with UX and performance improvements: theme and user preferences updates, per-topic indexing, documentation loading and search enhancements, SectMenu/sidebar improvements, and updated VPS deployment documentation and script. Compare: https://github.com/tommaso-berti/codexPane/compare/v2.6.0...v3.0.0

### New Features
- Enhanced theme management and user preferences: accent palette and reading time features (commit 48682ca).
- Added an index for each topic to improve webapp performance (commit 76838fd).
- Improved documentation loading and search via topic manifests and improved state management (commit 5b1063b).
- SectMenu extended with search and sidebar options persistence (commit ab14cc2).
- Updated deployment documentation and added a unified VPS deploy script (commit 3db5156).

### Bug Fixes
- No explicit bug fixes recorded in the commits for this range.

### Refactor & Engineering
- Generated manifests and topic indices updated: files under src/content/docs-topics/*.generated.json and src/content/docs-manifest.generated.json changed.
- Search and docs hooks improved: changes in src/features/search/*, useMiniSearchFromDocs.js, and src/features/docs/useDocContent.js.
- State management and UI preferences updated in src/store/uiPrefsSlice.js and src/store/store.js.
- UI components updated (SectMenu, Sidebar, SettingsDrawer, DocContent, Header, Layout, App — see src/components/*).
- New deploy script added: scripts/deploy-vps-unified.sh; VPS docs updated under src/content/vps/*.

### Breaking Changes
- No explicitly documented breaking changes are present in the commit metadata. Please review generated content files under src/content/ if you rely on them externally.

### Migration Notes
- If you consume generated files (src/content/docs-manifest.generated.json, src/content/docs-index.generated.json, or src/content/docs-topics/*.generated.json), update any pipelines or integrations that read them.
- Use scripts/deploy-vps-unified.sh and the updated docs under src/content/vps/* for the new VPS deployment flow.
- Verify any persisted UI preferences against changes in src/store/uiPrefsSlice.js.

### Known Limitations
- No known limitations explicitly documented in the commits for this release.
