## English

### Overview
- Release type: minor (payload.releaseType = "minor").  
- This release adds multiple interactive playgrounds for CSS, HTML and Bash, improves styling for table/playground components, refactors playground components to use a PlaygroundShell, updates documentation, and includes a targeted CSS fix.

- Added interactive CSS playgrounds for: box sizing, positioning, font stack readability, grid templates, button states, flex axes, transition timing, HSL controls, breadcrumb patterns, specificity, selectors, and responsive units. (c85e606)

- Updated demo section titles/icons and clearer interactive playground section titles. (ffcac9d, 003d531)
