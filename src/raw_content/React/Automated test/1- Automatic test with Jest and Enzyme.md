## 
## **Running packages**
* Using Yarn:
* yarn create-react-app my-app
* Using npm (specifically the <u>[npx command](https://docs.npmjs.com/cli/v8/commands/npx)</u> to execute a package without installing it):
* npx create-react-app my-app
## **Adding dependencies**
* Using Yarn:
* yarn add enzyme enzyme-adapter-react-16 --dev
* Using npm:
* npm install enzyme enzyme-adapter-react-16 --save-dev
## **Running package.json scripts**
* Using Yarn:
* yarn run test
* Using npm:
* npm run test
