# 1. Manipulating DOM


The console also recognizes <u>[$()](https://api.jquery.com/jquery/)</u> as an alias for <u>[document.querySelector()](https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelector)</u> to select an element. This shorthand resembles <u>[jQuery](https://www.codecademy.com/learn/learn-jquery)</u> syntax and helps make manipulating the DOM through the console even more efficient

```
$('body').style.color="blue"

```

$() syntax to select the <body> element and set the color property to "blue". This statement returns the value "blue" to the console, and the color change is immediately reflected on the webpage.
