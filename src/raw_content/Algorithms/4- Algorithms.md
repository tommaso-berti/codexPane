# 4. Algorithms


In programming, an *algorithm* is a series of step-by-step instructions that a developer gives to the computer to complete a task. All the code you’ve written so far has been algorithms!
Some problems are so common that there are dedicated and well-known algorithms. One such common problem is being able to sort data. There are quite a few well-known algorithms such as:
* <u>[Bubble Sort](https://en.wikipedia.org/wiki/Bubble_sort)</u><u>[Merge Sort](https://en.wikipedia.org/wiki/Merge_sort)</u><u>[Quicksort](https://en.wikipedia.org/wiki/Quicksort)</u>Since these algorithms are so well-known, they’re sometimes used as interview questions! Each algorithm eventually sorts an array of elements, but they all use different approaches. Don’t worry though, we’ll cover these algorithms and other helpful algorithms later on!
