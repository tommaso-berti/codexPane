# Semantic naming

Usa tag HTML semantici per migliorare accessibilità e SEO.

- `<header>`, `<main>`, `<footer>`
- `<nav>` per la navigazione
- `<section>` e `<article>` per strutturare i contenuti

```html
<main>
  <article>
    <h1>Titolo</h1>
    <p>Contenuto…</p>
  </article>
</main>
```

> Nota: nessun componente React dentro l’MDX, solo Markdown/HTML. Così compila senza import extra.
