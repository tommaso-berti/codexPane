# Js Singly **Linked Lists**

Let's implement a linked list in JavaScript. As you might recall, a linked list is a sequential chain of nodes. Remember that a node contains two elements:
* data
* a link to the next node
We are going to use a provided Node class, which you can find in Node.js. Make sure to use the proper Node methods throughout the lesson instead of accessing properties directly (ex., use someNode.getNextNode() instead of someNode.next).

Depending on the end-use of the linked list, there are a variety of methods that we can define. For our use, we want to be able to:
* add a new node to the beginning (head) of the list
* add a new node to the end (tail) of the list
* remove a node from the beginning (head) of the list
* print out the nodes in the list in order from head to tail

Linked list configuration example in JavaScript

```
const Node = require('./Node');

class LinkedList {

  constructor() {
    this.head = null;
  }

  addToHead(data) {
    const newHead = new Node(data);
    const currentHead = this.head;
    this.head = newHead;
    if (currentHead) {
      this.head.setNextNode(currentHead);
    }
  }

  addToTail(data) {
    let tail = this.head;
    if (!tail) {
      this.head = new Node(data);
    } else {
      while (tail.getNextNode() !== null) {
        tail = tail.getNextNode();
      }
      tail.setNextNode(new Node(data));
    }
  }

  removeHead() {
    const removedHead = this.head;
    if (!removedHead) {
      return;
    }
    this.head = removedHead.getNextNode();
    return removedHead.data;
  }

  printList() {
    let currentNode = this.head;
    let output = '<head> ';

    while (currentNode !== null) {
      output += currentNode.data + ' ';
      currentNode = currentNode.getNextNode();
    }

    output += '<tail>';
    console.log(output);
  }

}

module.exports = LinkedList;

```

## addToHead(data)

```
addToHead(data) {
  const newHead = new Node(data);
  const currentHead = this.head;
  this.head = newHead;
  if (currentHead) {
    this.head.setNextNode(currentHead);
  }
}

```

### How it works:
1. A new node is created with the given data.
2. The current head is saved in currentHead.
3. The new node becomes the new head.
4. If a head already existed, the new node points to the old head.
Effect: the new node is inserted at the beginning of the list.

## addToTail(data)

```
addToTail(data) {
  let tail = this.head;
  if (!tail) {
    this.head = new Node(data);
  } else {
    while (tail.getNextNode() !== null) {
      tail = tail.getNextNode();
    }
    tail.setNextNode(new Node(data));
  }
}

```

### How it works:
1. tail starts at the head.
2. If the list is empty, the new node becomes the head.
3. If the list is not empty:
	* Traverse the list until reaching the last node.
	* Set the last node's next pointer to a new node.
Effect: the new node is added at the end of the list.

## removeHead()

```
removeHead() {
  const removedHead = this.head;
  if (!removedHead) {
    return;
  }
  this.head = removedHead.getNextNode();
  return removedHead.data;
}

```

### How it works:
1. Save the current head in removedHead.
2. If the list is empty, stop execution.
3. Move the head pointer to the next node.
4. Return the data of the removed node.
Effect: removes the first node from the list.

## printList()

```
printList() {
  let currentNode = this.head;
  let output = '<head> ';

  while (currentNode !== null) {
    output += currentNode.data + ' ';
    currentNode = currentNode.getNextNode();
  }

  output += '<tail>';
  console.log(output);
}

```

### How it works:
1. Start from the head.
2. Traverse each node one by one.
3. Append each node's data to a string.
4. Stop when reaching null.
5. Add &lt;tail&gt; at the end and print the result.
Example output:

```
<head> 5 10 15 <tail>

```

## Overall Summary
This
     LinkedList
  class allows you to:
* Create an empty list
* Add nodes at the beginning
* Add nodes at the end
* Remove the first node
* Print the list in order
The list is called "singly linked" because each node only stores a reference to the next node, not the previous one.
